## Installation

### yarn

```bash
yarn add js-common-data-structures
```

### npm

```bash
npm install js-common-data-structures
```

## Binary Search Tree

```javascript
import { BinarySearchTree } from 'js-common-data-structures'

const tree = new BinarySearchTree()
tree.insert(12) // Add tree node
tree.preOrderTraverse(callback) // preOrderTraverse
tree.postOrderTraverse(callback) // postOrderTraverse
tree.inOrderTraverse(callback) // inOrderTraverse
tree.min() // find the minimum
tree.max() // find the maximum
tree.search(key) // find the tree key
tree.remove(key) // find the key
```
## AVL Tree

import { AVLTree } from 'js-common-data-structures'
